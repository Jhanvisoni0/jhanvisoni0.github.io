02 · SYSTEM DESIGN

Medallion Architecture in Practice // the pattern behind every platform I build

Pattern used across: Financial News RAG Platform, Retail Banking Analytics Platform, Real Assets Investment Intelligence Platform.

01

Raw Sources

APIs, streaming events, and source databases — ingested as-is via ADF, Auto Loader, and Kafka.

→
02

Bronze Layer

Immutable raw ingestion into Delta Lake — full history retained, schema captured as-is.

→
03

Silver Layer

Cleaned, deduplicated, conformed — validated against data quality rules, SCD Type 1/2 applied.

→
04

Gold Layer

Aggregated, business-ready star-schema tables — optimized via partitioning and Z-ordering.

→
05

Consumption

Power BI dashboards, ML models, executive reporting — access governed by Unity Catalog RBAC.

Core Principles

01

Schema Enforcement & Data Quality

Automated frameworks catch schema mismatches, nulls, and duplicates before they reach end users — monitoring 100+ feeds in production.

02

Incremental, Scalable Processing

Auto Loader and Delta Live Tables handle incremental ingestion without reprocessing full history.

03

Governance, Lineage & Security

Unity Catalog enforces table-level RBAC, Key Vault secures credentials, and every table's lineage stays traceable.
